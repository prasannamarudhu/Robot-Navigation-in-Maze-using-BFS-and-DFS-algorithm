Detailed Description

DFSAlgorithm class performs Depth First Search (DFS) operations on maze discovered in simulator.

DFSAlgorithm class is a derived class of Algorithm class, designed to analyze the maze file discovered from the simulator and find a path from the start location to goal location. The DFS algorithm will evaluate the precense of a wall to the neighboring cells to determine a clear path. It follows a path through depth iteration.
The accessible cell locations around the current position aree added to a queue of points to be evaluated.
This process continues for each search "branch" produced by the search algorithm until an evaluated point reaches the goal area.
As the search algorithm produces a path to the goal area, the robot is commanded to move through the visited point list towards the goal. Robot movement commands are acheived through the API commands in a separate function.
solve(): Checks if an evaluated point has reached the goal location and traces final path produced from DFS algorithm.
generatePath(): Runs DFS algorithm to search and evaluate points while producing a suitable path to the goal.
