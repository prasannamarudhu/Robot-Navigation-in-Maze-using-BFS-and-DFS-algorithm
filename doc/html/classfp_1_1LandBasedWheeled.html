◆ SpeedUp()

void fp::LandBasedWheeled::SpeedUp ( int  delta)

Function Prototyping.

FUNCTION ONLY IN LANDBASEDWHEELED.

The robot can increase it's speed, which is translated by the number of cells in the maze that the robot can traverse in each step.

SpeedUp(int): Increase speed==number of cells in the maze that the robot can traverse per step MoveForward(): Move the robot forward in the maze TurnLeft(): Turns robot 90 degrees counter-clockwise direction TurnRight(): Turns robot 90 degrees clockwise direction

Implements the speed increases in robot.

When SpeedUp is called, cout output is activated to identify SpeedUp.
